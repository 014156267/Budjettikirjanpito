### Budjettikirjanpito

Budjettikirjanpito on niin yrityksille, perheille kuin yksittäisille henkilöille suunnattu ohjelma, joka mahdollistaa kuukausittaisen rahankäytön seuraamisen. Ohjelmaan kirjataan tulojen ohella niin yksittäiset kauppareissut kuin suuretkin hankkeet: velat ja säästöt.
